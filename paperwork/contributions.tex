\title{\bf \Large 博士学位论文主要创新点概述}
\author{杨~帆\\ \small 中国科学院自动化研究所}
\date{\small 2014年9月17日}
\maketitle

本人所提交之博士学位论文《冠状动脉介入仿真中创建解剖环境的关键技术研究》研究创建人体解剖环境的虚拟模型的技术方法。
该文内容主要包括两方面：
一方面是冠状动脉介入导管术过程中所涉及的心血管系统的提取与可视化，包括：冠状动脉与主动脉组成的血管网络，冠状动脉围绕的心脏等；
另一方面是上述血管系统模型的数据优化和信息提取，包括：面向交互仿真的血管系统模型数据精简，血管系统模型的几何信息提取等。
其主要创新点可概括如下：

\begin{enumerate}
\item 针对手术仿真中人体主动脉的分割与可视化任务，我们提出了基于GAC的处理方法。
\item 针对手术仿真中人体冠状动脉树状结构的分割与可视化任务，我们提出了基于CURVES方法的处理方法。
\item 通过引入管状物增强技术改进上述基于CURVES方法的处理方法，以获得更完整的冠状动脉树状结构模型。
\item 为提高血管表面模型以及其它模型在手术仿真中的交互效率，对模型引入基于低通滤波器的数值的平滑处理，并在不破坏几何特征的前提下，对表面模型中多边形面片数量的精简。
\item 为提高虚拟解剖环境的真实性，我们提出了基于GAC的处理方法，获得了心脏的粗略表面模型。
\item 为提取血管表面模型的中心线等几何信息，我们提出了基于Voronoi图的类水平集演进的方法。
\item 为构建手术过程的模型，我们提出了基于有限状态机的建模方法的方法。
\end{enumerate}
