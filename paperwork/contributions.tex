\title{\bf \Large 博士学位论文主要创新点概述}
\author{杨~帆\\ \small 中国科学院自动化研究所}
\date{\small 2014年9月17日}
\maketitle

本人所提交之博士学位论文《冠状动脉介入仿真中创建解剖环境的关键技术研究》研究创建人体解剖环境的虚拟模型的技术方法。
该文内容主要包括两方面：
一方面是冠状动脉介入导管术过程中所涉及的心血管系统的提取与可视化，包括：冠状动脉与主动脉组成的血管网络，冠状动脉围绕的心脏等；
另一方面是上述血管系统模型的数据优化和信息提取，包括：面向交互仿真的血管系统模型数据精简，血管系统模型的几何信息提取等。
其主要创新点可概括如下：

\begin{enumerate}
\item 针对手术仿真中人体主动脉的分割与可视化任务，我们提出了基于测地活动轮廓（GAC）的处理方法。
该方法首先对原始CTA影像数据进行平滑处理，然后分别进行特征图像运算和初始水平集的生成，然后利用GAC方法提取目标的边缘，最后得出目标的表面模型。
\item 针对手术仿真中人体冠状动脉树状结构的分割与可视化任务，我们提出了基于CURVES方法的处理方法。
该方法首先对原始CTA影像数据进行区域提取与平滑处理，然后进行阈值处理，再分别进行特征图像运算和初始水平集的生成，然后利用CURVES方法提取目标的边缘，最后得出目标的表面模型。
\item 通过引入管状物增强技术改进上述基于CURVES方法的处理方法，以获得更完整的冠状动脉树状结构模型。
该方法首先对原始CTA影像数据进行区域提取与二值阈值处理，然后进行特定横截面积的管状物体的增强，并通过二值阈值滤波获得基本形状信息，再分别进行特征图像运算和初始水平集的生成，然后利用CURVES方法提取目标的边缘，最后得出目标的表面模型。
\item 为提高虚拟解剖环境的真实性，我们提出了基于GAC的处理方法，获得了心脏的粗略表面模型。
该方法首先对原始CTA影像数据进行区域提取与平滑处理，然后进行二值阈值处理得到心脏的近似区域，然后分别进行特征图像运算和初始水平集的生成，然后利用GAC方法提取目标的边缘，最后得出目标的近似表面模型。
\item 为提高血管表面模型以及其它模型在手术仿真中的交互效率，在不破坏几何特征的前提下，对表面模型中多边形面片数量的精简。
该方法首先对血管系统表面模型中的所有顶点的连接性进行检测以确保模型表面的完整性，然后引入基于低通滤波器的数值的平滑处理，最后再对表面模型中多边形面片数量的精简。
该方法的结果成功地提高了交互仿真的运行效率。
\item 为提取血管表面模型的中心线等几何信息，我们提出了基于Voronoi图的类水平集演进的方法。
该方法首先检测血管系统表面模型中的所有顶点的连接性，确保模型表面的完整性，以保证中心线计算的唯一性；
然后进行模型表面局部的平整处理，除去噪声干扰，保证中心线计算的准确；
其后进行模型表面的全局细分处理，确保中心线的数值计算过程保持稳定；
最后再构建表面模型内部空间的Voronoi划分，并基于此划分进行类似水平集演进的的计算，得到血管模型的中心线。
\item 为构建手术过程的模型，我们提出了血管介入手术过程的状态化建模方法。
该模型立足于血管模型中心线上的状态检测点，通过虚拟工具与这些点的碰撞检测来探知虚拟操作的进程。
\end{enumerate}
