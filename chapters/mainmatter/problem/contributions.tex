
\section{本文贡献}
\label{sec:contributions}

本文所述工作研究创建人体解剖环境的虚拟模型的技术方法。
在本文工作中，创建解剖环境就是利用图像处理，计算机图形学等学科的技术，以真实病例的CT血管造影（即：computed tomography angiography，简称CTA）数据为基础，重建并优化各主要人体解剖结构的几何模型，最终为使用仿真系统的用户提供足够真实的视觉感受。
本文内容主要包括两方面：
一方面是冠状动脉介入导管术过程中所涉及的心血管系统的提取与可视化，包括：冠状动脉与主动脉组成的血管网络，冠状动脉围绕的心脏等；
另一方面是上述血管系统模型的数据优化和信息提取，包括：面向交互仿真的血管系统模型数据精简，血管系统模型的几何信息提取等。
其主要贡献可概括如下：

\begin{enumerate}
\item 针对手术仿真中人体主动脉的分割与可视化任务，我们提出了基于测地活动轮廓（GAC）的处理方法。
该方法首先对原始CTA数据进行平滑处理，然后分别进行特征图像运算和初始水平集的生成，然后利用GAC方法提取目标的边缘，最后得出目标的表面模型。
\item 针对手术仿真中人体冠状动脉树状结构的分割与可视化任务，我们提出了基于CURVES方法的处理方法。
该方法首先对原始CTA数据进行区域提取与平滑处理，然后进行阈值处理，再分别进行特征图像运算和初始水平集的生成，然后利用CURVES方法提取目标的边缘，最后得出目标的表面模型。
\item 我们通过引入管状物增强技术改进上述基于CURVES方法的处理方法，获得了更完整的冠状动脉树状结构模型。
该方法首先对原始CTA数据进行区域提取与二值阈值处理，然后进行特定横截面积的管状物体的增强，并通过二值阈值滤波获得基本形状信息，再分别进行特征图像运算和初始水平集的生成，然后利用CURVES方法提取目标的边缘，最后得出目标的表面模型。
\item 为提高虚拟解剖环境的真实性，我们提出了基于GAC的处理方法，获得了心脏的粗略表面模型。
该方法首先对原始CTA数据进行区域提取与平滑处理，然后进行二值阈值处理得到心脏的近似区域，然后分别进行特征图像运算和初始水平集的生成，然后利用GAC方法提取目标的边缘，最后得出目标的近似表面模型。
\item 为提高血管表面模型以及其它模型在手术仿真中的交互效率，我们在不破坏几何特征的前提下，对表面模型中多边形数量的精简（达$99 \%$）。
该方法首先对已经获得的血管系统表面模型中的所有顶点的连接性进行检测以确保模型表面的完整性，然后引入基于低通滤波器的数值的平滑处理，最后再对表面模型中多边形进行精简。
该方法的结果成功地提高了交互仿真的运行效率（超过$10$fps）。
\item 为提取血管表面模型的中心线等几何信息，我们提出了基于Voronoi图的类水平集演进的方法。
该方法首先检测已经获得的血管系统表面模型中的所有顶点的连接性，从而确保模型表面的完整性，进而保证中心线计算的唯一性；
然后进行模型表面局部的平整处理，除去噪声干扰，进一步保证中心线计算的准确性；
其后进行模型表面的全局细分处理，以确保中心线的数值计算过程保持稳定；
最后再构建表面模型内部空间的Voronoi划分，并基于这一划分进行类似水平集演进的的计算，得到血管模型的中心线。
该中心线模型将为未来的工作提供基础信息。
\end{enumerate}
