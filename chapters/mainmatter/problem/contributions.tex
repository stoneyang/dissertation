
\section{本文贡献}
\label{sec:contributions}

本文所述工作研究创建冠状动脉介入导管术过程中所涉及的人体解剖环境的虚拟模型的方法，为仿真系统的解剖环境创建功能的实现提供关键技术支持。
冠状动脉介入导管术过程所处的解剖环境包括冠状动脉与主动脉组成的血管网络、冠状动脉围绕的心脏等。
在本文工作中，创建解剖环境就是利用图像处理、计算机图形学等学科的技术，以真实病例的CT血管造影数据为基础，重建解剖环境中各主要人体解剖结构的几何模型，最终为使用仿真系统的用户提供足够真实的视觉感受。

本文的主要贡献有：
\begin{enumerate}
\item 针对手术仿真中人体主动脉的分割与可视化任务，我们提出了基于测地学活动轮廓方法（Geodesic Active Contours，简称GAC）的处理方法。
\item 针对手术仿真中人体冠状动脉树状结构的分割与可视化任务，我们提出了基于CURVES方法的处理方法。
\item 为提高血管表面模型以及其它模型在手术仿真中的交互效率，对模型引入基于低通滤波器的数值的平滑处理，并在不破坏几何特征的前提下，对表面模型中多边形面片数量的精简。
\item 为提取血管表面模型的中心线等几何信息，我们提出了基于Voronoi图的类水平集演进的方法。
\item 为提高虚拟解剖环境的真实性，我们提出了基于测地学活动轮廓方法（GAC）的处理方法，获得了心脏的粗略表面模型。
\end{enumerate}
