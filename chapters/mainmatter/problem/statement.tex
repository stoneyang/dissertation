
\section{问题声明}
\label{sec:problem}

\subsection{问题描述}
\label{subsec:statement}

经皮冠状动脉介入术仿真训练系统（文中简称“仿真系统”或“训练系统”，不加区分），是以实现主要功能的软件系统为核心，辅以机电一体化的触觉操作机构而成的一个功能完整的混合系统。
设计并实现该系统的目的是，在有限的物理条件下，为受训者提供足够的真实性，从而使受训者能够“沉浸”在系统所提供的场景中，按照手术程序进行仿真实践，通过这一过程进一步了解手术流程，进而获得对手术技术更为形象的认识。

与实现训练系统有关的关键技术包括：解剖环境的创建、虚拟手术工具的创建、触觉接口的设计与制造、以及各部分工作的整合等方面。
本文的研究重点是解剖环境的创建，其中包括以下几个问题：
\begin{itemize}
\item 血管系统几何模型的建立；
\item 心脏可视化模型的建立；
\item 手术仿真的交互效率的提高等。
\end{itemize}

其中，
血管几何模型的主要作用是：为使用者提供在视觉上足够准确的与冠脉介入术相关血管系统的模型。
心脏模型需要满足的主要作用是：为使用者提供在视觉上足够提供冠状动脉各脉络的空间位置关系的心脏外围轮廓的近似模型。
在解剖环境模型与虚拟工具的交互中，为提高系统的实时性能，这两者在建模的过程中还需要对模型本身进行改进才能达到实际需求。
为实现训练系统的仿真功能，虚拟工具与解剖环境模型之间的交互过程中的运算效率尤为重要。

\subsection{设计需求}
\label{subsec:requirements}

基于第\ref{subsec:statement}节的讨论，我们列出实现训练系统的解剖环境模型的一些主要需求：
血管几何模型需要满足的基本要求是：尽可能准确地分割并重建出真实病例的血管系统模型，且能够比较准确地反映在冠脉介入术中涉及的主动脉、冠状动脉等血管的几何结构。
心脏模型需要满足的基本要求是：分割并重建出相应病例数据中的心脏外围轮廓的模型，并与冠状动脉几何模型实现准确的叠加。
手术仿真的交互效率的提高的目标是：在不明显改变血管模型的几何特征的前提下，通过最大限度的削减构成模型的数据，达到显著提高虚拟工具与血管模型的交互效率。
