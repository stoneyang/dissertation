
\section{展望}
\label{sec:conclusions_outlook}

\begin{enumerate}

\item \textbf{手术仿真中可视化效果的增强}

手术仿真的增强性效果的主要作用是：在交互过程中，表现一些重要的手术操作所产生的重要变化，从而使操作所产生的影响更加形象。
手术仿真的增强性效果的研究目标是：实现一系列视觉特效，以表现仿真训练过程中的一些视觉效果，从而改善仿真的真实性。
\begin{itemize}
\item 真实导管术中的C型臂X光投影仪的成像效果。
\item 血管模型应具备与真实血管相近的弹性和形变特性。
\item 所重建的心脏区域模型可以虚拟心脏的搏动过程。
\item 实现虚拟造影剂，该特性在触觉操作机构的激发下呈现，应当具有与真实手术过程中，医生通过导管向冠状动脉中注入造影剂时的视觉效果应当一致。
\item 实现虚拟支架-气囊结构，使其能够像真实情况一样――在气囊内部被气体填充的过程中，气囊不断膨胀，支架的结构随之发生变化（内径逐渐增大）；而在完成支架的安放后，气囊内部的气体被抽出人体外，气囊逐渐缩小至原状。
\item 人体的正常生理运动，如呼吸所引起的骨骼的运动等。 
\end{itemize} 

\item \textbf{手术过程的状态化建模}

基于血管模型的中心线来布设状态检测点（事实上是小球），并给这些状态点赋予碰撞检测功能，以此来探测虚拟手术工具是否经过该点所在，从而得出虚拟工具在血管模型中的行进位置。

\item \textbf{手术仿真训练的教学功能}

\begin{itemize}

\item 虚拟操作过程的记录

在操作开始后，训练系统能完整地记录使用者的操作过程。
这项功能可供使用者在操作结束后进行回放，以便回忆，纠正或复习。

\item 虚拟训练难度的分级

系统能够为使用者提供基于足够丰富的病例数据建立起来的若干组不同的虚拟解剖结构。
这些病例数据可以根据心血管疾病的不同发展阶段、病灶位置等来进行分类。
这样就实现了虚拟训练难度的区分，方便各种水平的使用者选择适合自己的病例进行虚拟训练。
使用者可以根据自身训练需要来选择相应的病例的虚拟解剖结构来进行仿真训练。

\item 虚拟训练的评价

训练过程的评价对于使用者的进步很重要，客观的评价能够使其更加明确自己的问题所在。
在上述两种功能的基础上，我们进一步的设想是，通过某种方法由系统为使用者的操作过程进行初步评价。
一种可能的实现方式是利用机器学习的技术方法，基于使用者操作的历史数据进行分析，得出较为符合客观的评价标准体系（即训练集）。
新的使用者的某次操作过程将会被归入某个分类之中，该分类对应的评价即为该使用者此次虚拟操作过程的一个初步评价。

\end{itemize}

\end{enumerate}
