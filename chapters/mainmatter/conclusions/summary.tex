
\section{总结}
\label{sec:conclusions_summary} 

本文所述工作研究创建人体解剖环境的虚拟模型的技术方法，主要是针对冠状动脉介入导管术过程中所涉及的心血管系统，包括：冠状动脉与主动脉组成的血管网络，冠状动脉围绕的心脏等。
在本文工作中，创建解剖环境就是利用图像处理，计算机图形学等学科的技术，以真实病例的CT血管造影数据为基础，重建并优化各主要人体解剖结构的几何模型，最终为使用仿真系统的用户提供足够真实的视觉感受。

本文的主要贡献有：
\begin{enumerate}
\item \textbf{人体心血管系统表面模型数据的提取}
\begin{itemize}
\item 针对手术仿真中主动脉的分割与可视化任务，我们提出了基于GAC的处理流程。
首先对原始医学体数据进行ROI提取和平滑等处理。
然后计算体数据的特征图像和初始水平集围线。
前者通过先求取各体素的梯度，再进行体素亮度的非线性映射来生成；
后者通过先进性阈值滤波除去无关内容细节，再利用快速行进算法进行水平集演进运算来获得初始水平围线。
其后引入GAC进行目标边缘的探测，对所得结果再进行二值滤波处理即得到目标的边缘信息。
最后引入行进立方体算法提取这些边缘信息用于表面的绘制，得到主动脉内腔表面的三维模型。
对结果的分析表明，该处理流程具有较高的准确性，足以满足以手术仿真为应用目的的需要。
\item 针对手术仿真中冠状动脉树状结构的分割与可视化任务，我们提出了基于CURVES的处理流程。
首先对原始医学体数据进行ROI提取和平滑等处理。
然后通过阈值滤波除去无关内容细节。
接下来计算体数据的特征图像和初始水平集围线。
其后引入CURVES进行目标边缘的探测，对所得结果再进行二值滤波处理即得到目标的边缘信息。
最后引入行进立方体算法提取这些边缘信息用于表面的绘制，得到冠状动脉空间树状结构的表面模型。
\item 通过引入管状物增强技术改进上述基于CURVES的处理流程，以获得更完整的冠状动脉树状结构模型。
在对原始医学体数据进行ROI提取和平滑等处理，通过二值阈值滤波除去无关内容细节，提高目标亮度所在范围内容的亮度。
接着利用多尺度线型滤波来增强体数据中的与冠状动脉横截面直径相近的所有管状物体的信息。
之后再次通过二值阈值滤波来提高增强结果中管状物体的亮度，同时除去无关内容细节。
其后计算体数据的特征图像和初始水平集围线。
再引入CURVES进行目标边缘的探测，对所得结果再进行二值滤波处理即得到目标的边缘信息。
最后引入行进立方体算法提取这些边缘信息用于表面的绘制，得到冠状动脉空间树状结构的表面模型。
通过与之前结果的对比可知，使用这种方法可以得到脉络细节更为完整的树状结构。
\item 为便于在虚拟解剖环境中判断冠脉分支，我们提出了基于GAC的处理流程，获得了心脏的粗略表面模型。
首先对原始医学体数据进行ROI提取和平滑等处理。
然后通过二值阈值滤波除去无关内容细节，提高目标亮度所在范围内容的亮度。
其后计算体数据的特征图像和初始水平集围线。
接着引入GAC进行目标边缘的探测，对所得结果再进行二值滤波处理即得到目标的边缘信息。
最后引入行进立方体算法提取这些边缘信息用于表面的绘制，最终得到近似的心脏表面三维模型。
从此处结果与冠脉模型的叠加效果可知，该方法所得心脏近似模型满足手术仿真的需要。
\end{itemize}

\item \textbf{面向交互仿真的人体心血管模型的数据可视化处理}
\begin{itemize}
\item 为提高血管表面模型与虚拟导丝在手术仿真中的交互效率，对模型引入基于低通滤波器的数值的平滑处理，并在不破坏几何特征的前提下，对表面模型中多边形面片数量进行精简。
为确保所处理表面的完整性和非冗余性，利用节点连接性验证算法来验证图像处理所得表面模型中的全部顶点之间的连接性，剔除多层或者游离状态的节点及其围成的多边形。
然后引入基于离散图信号的低通滤波器来平整表面中因噪声和图像处理所引入的尖刺部分。
最后引入多边形精简算法，在表面模型原有几何特征的约束下，充分精简表面模型中多边形的数量。
利用此结果进行交互仿真实验的结果表明，对于局部区段的表面模型，精简$90\%$的结果既满足视觉上的要求，又能满足仿真中对fps的最低要求（$> 10$）；
而对于完整的表面模型（包含完整的主动脉和冠状动脉），由于数据量过于庞大，精简$99\%$的结果才能满足仿真中对fps的要求，保证显示效果的流畅。
\item 为提取血管表面模型的中心线等几何信息，我们提出了基于Voronoi \\图的类水平集演进的处理流程。
为确保所处理表面的完整性和非冗余性，利用节点连接性验证算法来验证图像处理所得表面模型中的全部顶点之间的连接性，剔除多层或者游离状态的节点及其围成的多边形。
由于基于Voronoi图的提取方法对输入噪声敏感，故引入基于离散图信号的低通滤波器来平整表面中因噪声和图像处理所引入的尖刺部分。
接着，为保证类水平集演进算法的计算稳定性，我们引入改进的Butterfly策略对平整后的表面进行细分。
最后对细分后的表面模型的内部空间进行Voronoi划分，并在此基础上进行类水平集演进，最终得到血管表面模型的中心线。
\end{itemize}

\end{enumerate}
