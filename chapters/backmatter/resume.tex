\begin{resumesection}{科研经历}
\begin{itemize}
\item 2011~年~1~月至~2013~年~12~月，参与实验室承担的北京优博导师科技项目《微创血管介入手术操作培训的人机交互方法研究》（项目编号：Y1G2041FD1）。
该项目旨在研究并开发微创冠状动脉介入手术的计算机仿真系统, 用于为医学教研单位提供一种训练和研究手术技术的平台, 降低手术初学者和新手术技术对患者的不必要伤害。
在本项目中，本人所承担的职责有：
\begin{itemize}
\item 独立完成项目的前期调研
\item 完成原始样机的设计与实现
\item 负责仿真系统中基于~CT~影像的心血管系统建模
\item 合作完成心血管模型与虚拟导丝的交互实验
\item 设计并开发医学影像处理软件
\end{itemize}
\item 2009~年~6~月至~2012~年~12~月，参与实验室承担的国家“863”计划项目《微创血管介入手术机器人》（项目编号：Y0H2021F91/Y1H2021F91）。
该项目旨在研究并实现用于微创冠状动脉介入手术的机器人系统, 目的是降低医护人员的工作负担和不必要伤害, 提高手术微操作的精度和稳定性, 目前该系统已经完成了两轮迭代开发, 进行了四轮共~16~次临床活体动物实验。
在本项目中，本人所承担的职责有：
\begin{itemize}
\item 负责机器人控制软件中机械臂位姿同步显示功能的设计与开发
\item 独立完成了模拟导管室的设计与布局
\item 参与动物活体临床实验的设计
\item 进行活体实验的实际操作及实验数据分析
\end{itemize}
\end{itemize}
\end{resumesection}
